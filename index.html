What are AI coding agents?

  • LLMs that can read, write, and run code
  • Operate in a loop: plan → act → observe → repeat
  • Access to tools: file system, terminal, browser, APIs
  • Examples: Claude Code, GitHub Copilot agent, Cursor, Windsurf

Why Claude?

  • State-of-the-art on coding benchmarks (SWE-bench, LiveBench, etc.)
  • Claude Code — a CLI and VS Code extension by Anthropic
  • Agentic: reads files, runs commands, searches the web, makes commits
  • Integrates with GitHub, MCP servers, and custom tools
